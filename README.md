# point_tracker_bot

## About
Add and track points from completed quests on Telegram! An online alternative to pen & paper or excel spreadsheets. Useful for large-scale events like Amazing Race, Treasure Hunts, Escape Rooms and more!
Live example is available [here](https://web.telegram.org/#/im?p=@angelsAmongUsBot).

## Features
1. Track points for quests completed
2. Interact with developer for technical issues
3. Check total amount of points
4. Admin Features
- Display list of users with their corresponding points from highest to lowest
- Add new tasks
- Modify existing tasks (Password, Number of Points earned)
- Delete tasks
5. Super-Admin Features
- Add / Remove Admins

## Infrastructure
1. Database - MongoDB
2. Server - Heroku
